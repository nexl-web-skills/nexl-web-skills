Manifesto

nexl-web-skills · 内容质量门禁 · 四重审判循环

This repository is not a project. It is a statement. The author stopped caring about recognition. The account is a throwaway. The code is the only thing that speaks.

The four trials

TRIAL 01

Mediocrity purge

If a mid-level frontend dev could write it, delete it. No filler, no obvious comments, no framework template residue. Precise like a scalpel.

TRIAL 02

Ruthless aesthetics

No welcome. No emoji. No exclamation marks. The first sentence hits the pain point. Cold English, or minimal bilingual. Power needs no punctuation to be felt.

TRIAL 03

Architectural disrespect

Zero redundant dependencies. Brutal minimalism. Obsessive control over the browser's底层. When they read the source they think: "wait, it can be written like this?" Not human intuition. Better.

TRIAL 04

Anonymous god vibe

Commits are cold, minimal, direct. No apologies. No hand-holding in issues. Code is the only passport.

Run the four trials on everything you add. Survive, then ship.
