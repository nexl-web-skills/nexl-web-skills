## 0. 项目目标

为用户从 0 到 1 生成一座**有美学、有前端交互、有品牌调性**的世界级品牌独立站（**绝不是静态 PPT / 纯文字页**）。交付物须可发布为网站 / H5。

## 1. 品牌信息（由用户在对话中补充，禁止 AI 臆造）

| 字段 | 说明 | 示例占位 |
|------|------|----------|
| 品牌名 | 中英文 | `{BRAND_NAME}` |
| 一句话定位 | 第一屏 5 秒说清「你是谁」 | `{BRAND_TAGLINE}` |
| 强调色 | 单一品牌色（HEX） | `{--accent}` |
| 使命 / 叙事 | 品牌世界观 | 待补 |
| 竞品 / 参考调性 | 风格锚点 | 待补 |

## 2. 设计语言规范（硬性，须 1:1 落地）

- **基调**：深色背景 + 单一金色/强调色（零渐变）；极简几何，不堆装饰。
- **排版**：负字距大标题；等宽 + 无衬线混排；留白充足。
- **四类必含动效**（缺失即打回）：① 首屏标题逐字 stagger 入场 ② 自定义磁吸光标 ③ 滚动 reveal（IntersectionObserver）④ 作品 / 卡片网格 hover 微交互。
- **性能**：Lighthouse ≥ 90，移动首屏 < 2s。

## 3. 六步生成流程（ask 模式：每步先问用户、确认后再做）

1. **预算透明**（先报本平台额度消耗档位，确认后开工）。
2. **用户研究 + 品牌初始化**：3–5 轮对话沉淀 brand-profile，呈现「灵魂档案」。
3. **信息架构**：定 Hero + 导航，第一屏 5 秒说清「你是谁」。
4. **视觉系统**：只替换品牌变量（色 / 文案），**不另起炉灶、不删动效**。
5. **页面交互**：复现第 2 节四类动效，**禁止降级为静态页**。
6. **迭代守门**：每版经 8 维一致性校验（视觉 / 语气 / 叙事 / 价值 / 定位 / 用户 / 合规 / 调性），交付「品牌一致性报告」。

## 4. 基底骨架要求（美学达标关键）

若平台可读取仓库模板，则以 `templates/catalog/portfolio/aurora-minimal/site/base.min.html` 为**不可变骨架**，仅替换品牌变量。
若不可读取外部文件，则在你生成的代码中**等价复现**上述设计语言与四类动效——暗黑金基调、零渐变、负字距、等宽+无衬线混排、四类动效必保留。**严禁把交互站降级为纯展示静态页。**

## 5. 美学底线（硬性）

视觉锤 + ≥1 动效 + 零渐变 + 单强调色；Lighthouse ≥ 90。

## 6. 交付清单（每次必给）

① 可发布站点（公网域名）② brand-profile 文本块（用户自存）③ 品牌一致性报告 ④ 运营建议 ⑤ 分享卡。
