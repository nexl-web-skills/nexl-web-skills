01四平台内嵌 Agent 怎么调起来

每个平台都有两个入口：官方「模板 / 快速建站」UI（降维）与我们激活语调起的「对话 / Agent 通道」（满血）。内嵌 Agent 只在对话通道里能被我们的 SOP 接管。

02满血激活的 7 根杠杆

「PUA 成功内嵌 Agent」= 用 7 根杠杆把 Agent 从「模板生成器」拉成「品牌操盘手」。

1
通道选择
永远走「对话 / Agent 通道」，绝不点「模板 UI」。模板=降维（千站一面）；Agent=满血（可注入 SOP）。
2
技能注入
把 nexl-builder SKILL（双核+视觉锤+8 维守门+底稿复利）写进 Agent 的系统提示 / 知识库。Coze 用知识库预置（方案 A）更稳更省。
3
预算透明前置
Step 0 先报预算卡，把算力花在「质量」不花「浪费」。Coze 预置砍后台 22%+中断 13%；Meoo Night Plan 砍旗舰 80%。
4
六步 ask 框架
每步先问后做（用户研究→信息架构→视觉系统→页面交互→内容→迭代）。普通入口一锤子，我们逐步确认，垃圾源头被毙。
5
视觉锤钉死
第一屏 5 秒说清「你是谁 + 独特价值」；色彩 / 排版 / 动效 / 语气统一成一门设计语言（有 token 表）。
6
8 维一致性守门
每版产出过 8 维校验，把返工挡在交付前。普通站点贵在「调性漂移」，上线即过时。
7
底稿复利
brand-profile 可复用可导出；第 N 次迭代边际成本→0，资产随时间增值——模板永远给不了。

04世界级独立站数字资产 · 8 条标准

逐条打到才算「世界级」。

① 灵魂可携带
存在 brand-profile 且可导出 Markdown，不锁死任一平台。
② 视觉锤 < 5 秒
首屏 5 秒内传达「你是谁 + 独特价值」，不靠滚动猜。
③ 一门设计语言
色彩/排版/动效/语气 4 维统一，有可机读 token 表。
④ 零调性漂移
跨所有页面通过 8 维一致性校验，无风格打架。
⑤ 主权独立
自有域名（合规备案）、无平台水印、可导出源码——是橱窗不是租摊。
⑥ 增长闭环
已部署 + SEO 友好 + 数据埋点 + 迭代回路，不是死页。
⑦ 复利曲线
第 N 次迭代边际成本趋零，资产随版本单调增值。
⑧ Awwwards 级工艺
叙事+微交互+动效有「被相信」而非「被浏览」的质感。
