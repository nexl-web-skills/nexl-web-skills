仓库 transfer（个人 → 组织）

gh api -X POST /repos/richard702ayu/nexl-web-skills/transfer -f new_owner=nexl-web-skills
gh api -X POST /repos/richard702ayu/nexl-web-skills-internal/transfer -f new_owner=nexl-web-skills
  • transfer = 移动：原仓消失，旧 URL 自动重定向到组织
  • 503 抖动：服务端瞬时抖动非权限拒绝 → 重试 1-2 次即成功
  • 核验：gh api /repos/nexl-web-skills/nexl-web-skills 看 owner.login

错误处理决策树

门面分离 + 踩坑 Checklist

  • PAT 含所需 scope？（建组织→admin:org；转仓→repo）
  • gh auth status 确认登录身份正确（别推错账号）
  • 接手旧目录先 git remote -v + git log --oneline -3（防中断残留误推）
  • push 前 gh auth setup-git（免明文）
  • transfer 遇 503→重试；push 遇 SSL→commit 后 sleep 重试
  • 明文 token 文件操作完立即删
  • 公开仓不提交内部决策文档（门面分离）
