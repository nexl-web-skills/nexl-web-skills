部署 SOP

把 tideshellbase 部署到 CloudBase 静态托管默认域名，供国内用户内测。配套 hosting-no-filing.md。

1

登录锁定：tcb login → tcb env use cloudbase-d9gumwykf42c90a92 → tcb env list 确认。

2

开通托管：tcb hosting detail -e ... 首次自动开通静态网站托管。

3

部署：tcb hosting deploy ./nexlbase/tideshellbase /tideshell -e ... --yes

4

拿默认域名：tcb hosting detail 输出形如 cloudbase-d9gumwykf42c90a92.ap-shanghai.app.tcloudbase.com，记为 DEFAULT_DOMAIN。

5

回填占位：把 ACTIVATION.md/.html 与 install.sh 里的 {DEFAULT_DOMAIN} 替换成真实域名。

6

验证：tcb hosting list 见 /tideshell 文件；浏览器访问弹中间页 → 点确认 → 可访问。

7

内测入口：范式 A 复制 SKILL.md 全文发给 Agent（最稳）；范式 B curl .../install.sh | bash（带兜底）。

关键命令

tcb hosting deploy ./nexlbase/tideshellbase /tideshell \ -e cloudbase-d9gumwykf42c90a92 --yes tcb hosting detail -e cloudbase-d9gumwykf42c90a92 # 拿默认域名 tcb hosting delete /tideshell --dir -e ... --yes # 回滚

部署后中间页会挡 Agent 自动抓取——内测先走「复制 SKILL.md」。
