# 三平台内嵌 Agent 满血调用战略 · 世界级独立站数字资产标准

## 〇、一句话战略

> **普通用户**：登录 → 点官方「建站」入口 → 选模板 → 一次性产出 → 上线即过时。
> **NEXL 用户**：发激活语 → 调起内嵌 Agent 加载 nexl-builder → 双核（品牌灵魂底稿 + 建站）→ 8 维守门 → 复利资产。
>
> 差距不在工具，在「是否让 Agent 满血跑了我们的 SOP」。

「PUA 成功平台内嵌 Agent」这句话的本质，不是操控，而是**把我们的 SOP 写进 Agent 的运行时上下文，逼它走双核 + 视觉锤 + 8 维守门，而不是走平台默认的「模板生成」降维通道**。下面分三块讲清：① 三平台各自怎么调起内嵌 Agent；② 满血激活的 7 根杠杆；③ 世界级独立站数字资产的可量化标准与达成路径。

## 一、三平台内嵌 Agent 怎么调起来（官方入口 vs NEXL 满血路径）

每个平台都**有两个入口**：官方「模板/快速建站」UI（降维）与我们激活语调起的「对话/Agent 通道」（满血）。平台的内嵌 Agent 只在**对话通道**里能被我们的 SOP 接管。

| 平台 | 官方入口（降维·别走） | NEXL 满血调起方式 | 内嵌 Agent 调用机制 | 计费锚点（实测） |
|------|----------------------|------------------|---------------------|----------------|
| **Coze（扣子）** | 「网页生成」模板插件 / 模板画廊 | 创建 **Bot** → 知识库**预置 builder.md**（方案 A 最稳）→ 人设写入双核亮相 + 六步摘要；或「工作流」串「读 builder.md→预算卡→六步」节点 | Chat API v3 + doubao 系列；无终端，curl\|bash 无效 | 15 轮≈**400 积分（¥0.4）**；含 4 图≈1280（¥1.3） |
| **秒悟 Meoo** | 模板市场 /「快速建应用」 | 对话首句**粘贴激活语** → Agent 读 builder.md 激活 nexl-builder | 对话通道 + qwen/kimi/glm/deepseek（**仅境内合规模型，禁接 OpenAI/Google**） | 15 轮 ¥0.14–2.3；**Night Plan**（22:00–08:00）qwen3.7-max **2 折≈¥0.10** |
| **腾讯 WorkBuddy** | 「AI 生成网站」一句话 / 默认生成 | **技能激活 nexl-builder**（观涛虾本地执行） | 本地桌面 Agent + 文件系统 + bash + present_files，积分走 CodeBuddy 共享池 | 免费 500 积分/月 ≈ **6 次**建站（15 轮≈75 积分 ¥0.075） |

**关键洞察**：官方入口是「模板引擎」，内嵌 Agent 是「可被 SOP 接管的推理体」。我们永远走 Agent 通道，且在第一句话就把 builder.md 注入它的运行时——这就是「调起」的全部秘密。

## 二、满血激活的 7 根杠杆（让 Agent 远超普通入口人群）

「PUA 成功内嵌 Agent」= 用 7 根杠杆把 Agent 从「模板生成器」拉成「品牌操盘手」：

1. **通道选择**：永远走「对话 / Agent 通道」，绝不点「模板 UI」。模板 = 降维（千站一面）；Agent = 满血（可注入 SOP）。
2. **技能注入**：把 `nexl-builder` SKILL（双核 + 视觉锤 + 8 维守门 + 底稿复利）写进 Agent 的系统提示 / 知识库。Coze 用**知识库预置**（方案 A）而非每次抓 URL，更稳更省。
3. **预算透明前置**：Step 0 先报预算卡，把算力花在「质量」而非「浪费」。Coze 知识库预置可砍掉后台 22% + 中断浪费 13%；Meoo Night Plan 把旗舰模型成本砍 80%。
4. **六步 ask 框架**：每步先问后做（用户研究→信息架构→视觉系统→页面交互→内容→迭代）。普通入口是一锤子产出，我们是逐步确认，垃圾在源头被毙。
5. **视觉锤钉死**：第一屏 5 秒说清「你是谁 + 独特价值」；色彩 / 排版 / 动效 / 语气统一成**一门设计语言**（有 token 表）。
6. **8 维一致性守门**：每版产出过 8 维一致性校验，把返工挡在交付前。普通站点贵在「调性漂移」，上线即过时。
7. **底稿复利**：`brand-profile` 可复用、可导出；第 N 次迭代边际成本→0，资产随时间增值——这是模板永远给不了的。

## 三、为什么远超普通入口人群（机制差，非运气）

| 维度 | 普通入口人群 | NEXL 满血用户 |
|------|-------------|---------------|
| 起点 | 套模板 | 品牌灵魂底稿（brand-profile） |
| 第一屏 | 业务罗列、千站一面 | 视觉锤 5 秒说清「你是谁」 |
| 设计语言 | 平台默认主题 | 一门统一语言（色/排/动/语） |
| 过程 | 一次性产出 | 六步 ask 逐步确认 |
| 质量阀 | 无 | 8 维一致性守门 |
| 资产形态 | 会过时的页面 | 复利底稿（可携带、可迭代） |
| 单次成本 | 外包 ¥2万–8万 / 模板年费 | **¥0.1–1.3 / 次** |

**结论**：普通人群买到的是「一页会过时的漂亮皮」；NEXL 用户买到的是「一份持续复利的品牌数字资产」。这不是话术，是成本结构与资产属性的根本不同。

## 四、世界级独立站数字资产 · 8 条可量化标准

如何定义「世界级」？用 8 条可验证标准，逐条打到才算：

1. **灵魂可携带**：存在 `brand-profile` 且可导出 Markdown 文本块，不锁死任一平台。
2. **视觉锤 < 5 秒**：首屏 5 秒内传达「你是谁 + 独特价值」，不靠用户滚动猜。
3. **一门设计语言**：色彩 / 排版 / 动效 / 语气 4 维统一，有可机读的 token 表。
4. **零调性漂移**：跨所有页面通过 8 维一致性校验，无风格打架。
5. **主权独立**：自有域名（合规备案）、无平台水印、可导出源码——是「橱窗」不是「租摊」。
6. **增长闭环**：已部署 + SEO 友好 + 数据埋点 + 有迭代回路，不是死页。
7. **复利曲线**：第 N 次迭代边际成本趋零，资产随版本数单调增值。
8. **Awwwards 级工艺**：叙事 + 微交互 + 动效有「被相信」而非「被浏览」的质感。

## 六、铁律（防黑盒 / 防降维）

- 激活语永远固定那一句，不为任何平台改文案。
- 永不走官方模板入口（那是降维通道）。
- 不报预算不许开工（Step 0 硬前置）。
- 8 维守门每版必过，返工挡在交付前。
- 底稿必导出「可复制文本块」供用户自存。
- 合规红线：Meoo 不接境外模型；域名必合规备案；不生成违规内容。
