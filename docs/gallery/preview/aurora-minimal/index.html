ABOUT

我相信好设计是被相信，不是被浏览。多数网站死在第一屏——拼命展示业务，却没回答你是谁。

我的顺序永远是反的：先钉死视觉锤，再让它长成系统。

页面会过时，那一份品牌底稿不会。换项目、换平台，骨架与声音都复用，边际成本趋零。
