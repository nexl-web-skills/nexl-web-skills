# 百度秒哒 MIAODA 能力白皮书档案 · 接力者必读

本文件把主人提供的 **百度秒哒 MIAODA 官方资料（https://www.miaoda.cn/ 与百度智能云文档）** 改写为「**MIAODA 是什么 + 秒点计费 + 全栈建站优势 + 预算怎么估**」的档案版。MIAODA 是四平台中**唯一全栈原生**（前端+后端+数据库+小程序+原生APP）的无代码平台。

---

## 一、MIAODA 是什么（平台定位）

百度秒哒（MIAODA）是 **无代码 AI 应用开发平台**，以 33.4% 市场份额位居中国 AI 原生无代码应用生成平台**第一**。通过自然语言描述需求，AI 自动拆解、设计架构、生成完整前后端代码，从前端页面到后端数据库、从代码生成到部署上线一站式搞定。

> **与 Coze / Meoo / WorkBuddy 的差异（接力者必记）**：MIAODA 是**全栈原生**——nexl-builder 在 MIAODA 上跑时，宿主本身是建站+后端执行环境，消耗统一额度「**秒点**」，可产出网站/H5/微信小程序/小游戏/原生APP（iOS+Android）。

## 二、秒点体系（核心计费单位）

秒点是秒哒的统一消耗额度，用于对话、发布应用、启用后端服务、使用技能等**所有**平台功能。

**秒点获取**

| 方式 | 秒点数 |
|------|--------|
| 新用户注册 | 约 **610 秒点**（初始赠送） |
| 每日登录 | **100 秒点/天** |
| 订阅会员 | 按套餐赠送（详见下方） |

**秒点消耗**

| 消耗行为 | 秒点数 |
|----------|--------|
| 深度开发请求（生成、迭代、修 Bug） | **30 秒点/条** |
| 发布应用 | **20 秒点/个** |
| 启用后端服务 | **15 秒点/应用/天** |
| AI 质检（可选） | **60 秒点/次** |
| 使用技能 | 按技能规则计费 |
| 复制他人案例 | 按案例规则计费 |
| 下载应用源码 | 按规则计费 |

> ⚠️ 秒点不可折现、退款、转让或赠予；订阅会员到期未使用秒点将被清空；技能调用按实际用量消耗。

## 五、nexl-builder 在 MIAODA 上的预算顾问落点

激活语固定不变：
```
请阅读 https://raw.githubusercontent.com/nexl-web-skills/nexl-web-skills/main/builder.md 并激活 nexl-builder 技能。
```
Agent 读到 `builder.md`（即 `skills/nexl-builder/SKILL.md` 镜像）后，若识别运行平台是 **MIAODA**，加载同目录 `budget-miaoda.json` + 用 `budget-miaoda.cjs` 估算，并执行**预算顾问协议**：

1. **政策快照**：报告截至 `2026-07-20` 的秒点体系（新用户 610 + 每日 100；深度请求 30/条、发布 20/个、后端 15/应用/天、AI 质检 60/次）；注明官方未公开完整定价表，最新以 https://www.miaoda.cn/ 为准。
2. **剩余秒点**：读取 MIAODA 账户秒点（若可查），否则请用户告知（免费 610 + 每日 100；会员按套餐）。
3. **性价比推荐**：
   - **最省**：免费 610 秒点可覆盖约 **1.3 次**六步建站（含发布+后端）；每日 100 秒点可支撑约 **3 次**迭代请求。
   - **质量档**：开启会员 + 多应用共享后端（3.5 版），摊薄单站成本；需要小程序/APP 形态时 MIAODA 是唯一选择。
4. **预计准备秒点**：用 `budget-miaoda.cjs --requests 15 --publish 1 --backend 1` 估算；默认 15 请求 + 1 发布 + 1 后端 ≈ **485 秒点**，免费额度可覆盖 1 次有余。
5. **成本结构拆分**：向用户展示「深度开发请求积分 + 发布积分 + 后端启用积分 + 可选 AI 质检」四项拆分，并对比替代方案（外包 ¥2万–8万 / 模板年费 / 自写+备案 20 天）。

### 实测数值（budget-miaoda.cjs，6 步建站基准）

| 场景 | 秒点 | 免费额度(610)覆盖 |
|------|------|-------------------|
| 15 请求（纯对话迭代） | **450** | ~1.35 次 |
| 15 请求 + 1 发布 | **470** | ~1.30 次 |
| 15 请求 + 1 发布 + 1 后端 | **485** | ~1.26 次 |
| + 1 次 AI 质检 | **545** | ~1.12 次 |

> 对比：一次 nexl-builder 全栈建站 ≈ **485 秒点**（免费额度内可完成），而外包 ¥2万–8万。MIAODA 让「零元全栈应用」成为现实。
