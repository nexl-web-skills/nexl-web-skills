## 执行摘要

nuwa-skill 用 ~106 天做到 **28,360 stars / 3,996 forks**，峰值日增超 4,000。它是"品类红利 + 病毒拐点 + 内容营销 + 生态固化"的经典 GitHub 爆款。但它的增长已入 S 曲线后半段（7月日均回落至 ~126），且存在 **6 个结构性弱点** 可被我们系统性超越。

本 Playbook 给出一套**世界级标准的裂变引擎**：8 根增长杠杆 + 1 个差异化飞轮（品牌经纪人留存），并全部落地到 nexlbase 仓库与在线 demo。核心一句话——

> **nuwa 让人"哇"一下就走；nexl 让每个品牌"建完即代言、用久即沉淀"。**

## 二、nuwa 做对的 8 件事（增长杠杆）

1. **品类红利卡位**：在"蒸馏人"被验证可行的窗口，用更强叙事抢占品类心智。
2. **低摩擦激活**：`npx skills add` 一行安装，跨 55+ runtime，采用门槛极低。
3. **效果可见即可传播**：README 堆满名人视角对话（乔布斯/马斯克/芒格），一看就 wow，天然截图裂变。
4. **质量可信度背书**：保真度评分卡（独立双 agent 盲测 A 级），解决"是不是噱头"的信任问题。
5. **UGC 飞轮**：COMMUNITY 索引让用户提交自己蒸馏的 skill（star 归用户），生态自增长。
6. **多语言覆盖**：中/英/日/韩/西 README，吃全球流量。
7. **内容引擎**：promo deck、hero 动画、quote cards 持续供给可分享物料。
8. **AI 辅助高频迭代**：单日 11 提交，用 Claude 快速打磨 SKILL.md 各维度。

## 三、nuwa 的 6 个可被超越的弱点

| # | 弱点 | 我们的机会 |
|---|------|-----------|
| 1 | **用完即走**：蒸馏名人是一次性娱乐/咨询，无持续使用钩子，留存弱 | nexl-broker 让 agent 成为"永远在线的品牌经纪人"，brand-profile 持续沉淀 → 留存天然强 |
| 2 | **传播素材静态**：示例对话是作者写的，用户不能一键生成自己的可分享物 | 建站产出即带回流水印的**分享卡**，用户每次建站都自动产生传播素材 |
| 3 | **无在线漏斗**：必须装 skill 才能体验，漏掉 90% 不装的人 | EdgeOne **在线 demo**，零安装即可试，漏斗大一个数量级 |
| 4 | **品类天花板**：名人蒸馏有伦理争议（公开≠真实）+ 名人有限 | 我们是"品牌资产化"，面向无数中小品牌/IP/创业者，市场大几个数量级 |
| 5 | **裂变无结构化激励**：社区贡献仅"star 归你"，无 referral | 设计**结构化裂变激励**（邀请建站得三平台积分/模板），映射到 Coze/Meoo 额度 |
| 6 | **地域集中**：起家中文圈（B站/小红书） | 我们已覆盖三平台（腾讯/百度/秒悟 + 可扩海外），"品牌出海"叙事天然全球化 |

## 四、nexl 病毒引擎：8 杠杆 + 1 飞轮

### 8 根增长杠杆（世界级标准）

**L1 · 零安装漏斗（最大差异化）**
EdgeOne 部署 `demo/` 单页：输入品牌名+一句话定位 → 实时生成品牌站预览 + 分享卡 + 激活语。不装 skill 也能 wow，体验即传播。这是 nuwa 没有的入口。

**L2 · 低摩擦激活（对齐 nuwa）**
双激活语：安装型 `npx skills add nexl-web-skills/nexl-web-skills`（短）+ 即载型固定句（Coze/Meoo 兜底）。

**L3 · 产出即传播（核心机制）**
`scripts/gen-brand-card.cjs`：建站完成时自动生成（a）带 nexl 回流链接的分享卡 SVG，（b）一段可一键转发的推文/朋友圈文案。用户发出去 = 免费代言。

**L4 · UGC 案例飞轮**
`showcase/` 目录 + `CONTRIBUTING.md`：用户建完站一键提交案例（star 归用户），形成"别人用 nexl 建了什么站"的活广告库。

**L5 · 品牌经纪人留存飞轮（差异化飞轮）**
nexl-broker 让 agent 成为用户的品牌守门人——品牌资产（brand-profile）越用越厚，切换成本越来越高，留存与口碑双增。

**L6 · 多语言 × 多平台**

**L7 · 结构化裂变激励**
邀请机制：老用户邀请新用户建站，双方得三平台积分/模板额度（映射到 Coze/Meoo 等）。开源无付费，但积分是真实货币。

**L8 · 内容引擎 + KOL 杠杆**
复用 nuwa 的"可截图素材"打法，但升级为"用户生成的真实案例"——每个 showcase 案例都是自带流量的内容。种子 KOL（品牌/出海/独立开发者圈层）首发体验。

### 差异化飞轮（为什么我们能超越留存）

```
用户建站 → 产出分享卡(裂变) → 案例进 showcase(社会证明)
   → brand-profile 沉淀(留存) → 下次建站更懂品牌(复利)
   → 更愿意邀请他人(激励) → 新用户进入漏斗 → 循环加速
```

nuwa 只有前半段（建完即走）；我们有完整闭环。
