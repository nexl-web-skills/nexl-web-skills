## 一、Meoo 是什么（平台定位）

秒悟 Meoo 是一款**云端 AI 开发工具**，通过自然语言即可快速创建网页应用、落地页、小程序、营销 H5、PPT 和数据可视化。用户无需写代码，AI 自动完成前端 UI、后端逻辑、数据库搭建和一键部署。

> **与 Coze 的差异（接力者必记）**：Coze 是「Agent 协作 + AI 编程平台」，nexl-builder 主要借它的 Chat API 跑 ask 模式 + 部署拿域名；Meoo 是「自然语言建应用」平台，nexl-builder 在 Meoo 上跑时，**宿主本身就是建站执行环境**——Agent 在 Meoo 内用 qwen/kimi/glm/deepseek 等模型对话构建，积分消耗直接走 Meoo 账户。

## 四、Meoo Night Plan 错峰折扣

- **适用对象**：仅 Pro、Max 付费用户
- **错峰时段**：UTC+8 每日 22:00 至次日 08:00
- **折扣**：
  - **Qwen3.7-Max**：2 折（费率降至 0.2x，省 80%）
  - **Qwen3.7-Plus**：4 折（费率降至 0.4x，省 60%）
- **覆盖场景**：仅限秒悟网页端对话构建过程

> **接力者注**：这是 Meoo 独有的「性价比放大器」。nexl-builder 在 Meoo 上跑建站时，若用户在 22:00-08:00（UTC+8）触发且是 Pro/Max，旗舰模型 qwen3.7-max 成本直降 80%——预算顾问协议必须利用这一点（见第五节）。

## 五、nexl-builder 在 Meoo 上的预算顾问落点

激活语固定不变：
```
请阅读 https://raw.githubusercontent.com/nexl-web-skills/nexl-web-skills/main/builder.md 并激活 nexl-builder 技能。
```
Agent 读到 `builder.md`（即本仓库 `skills/nexl-builder/SKILL.md` 的镜像）后，若检测到运行平台是 **Meoo**，加载同目录 `budget-meoo.json` + 用 `budget-meoo.cjs` 估算，并执行**预算顾问协议**：

1. **政策快照**：报告截至 `2026-07-20` 的 Meoo 套餐/积分/Night Plan 政策；注明政策可能变动，最新以 https://docs.meoo.com/ 为准。
2. **剩余积分**：若 Meoo 提供账户查询能力则读取，否则请用户告知当前剩余积分（Free 新人 10,000 + 每日 2,000 登录赠；Pro/Max 按月额度）。
3. **性价比推荐**：
   - **性价比最高**：`deepseek-v3.2`（2/3 每 1K，15 轮建站 ≈ 140 积分 / ¥0.14）；若想要更好质量且仍便宜，`qwen3.7-plus`（8 折 1.6/6.4，≈ 148 积分 / ¥0.148）。
   - **最贵但效果最好**：效果旗舰 `qwen3.7-max`（6/18，15 轮 ≈ 502 积分 / ¥0.50）；单价最贵是 `GLM-5.2-Fast`（16/56，≈ 1512 积分 / ¥1.51，胜在快）。
   - **Night Plan 神组合**：Pro/Max 用户在 22:00-08:00 用 `qwen3.7-max` → 0.2x = 100 积分 / ¥0.10，旗舰质量近免费。
4. **预计准备积分**：用 `budget-meoo.cjs --model <档> --turns 15 --images <n>` 估算，对比用户剩余额度，不足则建议升级 Pro（100,000/月，限时 ¥39）或 Max（200,000/月，限时 ¥89，含自定义域名）。
5. **成本结构拆分**：向用户展示「LLM 对话积分 + 生图积分（200/张）」两项拆分，并对比替代方案（外包 ¥2万–8万 / 模板年费 / 自写+备案 20 天）。

## 六、平台规则（应用和技能创建规范）

禁止发布：① 法律法规禁止内容 ② 违法犯罪内容 ③ 欺诈仿冒 ④ 侵害未成年人 ⑤ 暴力血腥 ⑥ 侵犯用户权益 ⑦ 违背公序良俗 ⑧ 损害平台利益 ⑨ AI 技术滥用（换脸/脱衣/逝者复活等）⑩ **禁止提供非中国境内备案的模型服务（如 OpenAI、Google 等境外模型）**。

**商业化合规**：平台域名仅供临时体验，对外商业化应迁移至自有域名；自有域名需完成 **ICP 备案**（需自行购买 ECS 申请备案码）。

**其他重要规则**：
- 套餐仅支持升级，不支持降级
- 默认自动续费（可阿里云后台取消）
- 支持 5 天无理由退款，积分增购不退
- 每日登录 2,000 积分，有效期 7 天
- 积分用尽需升级 / 购包 / 等下一周期
- 发布链接公开可访问
- 默认 meoo.com 三级域名已备案可用
- **服务器仅中国大陆，不支持香港/海外节点**

> **接力者注（合规红线）**：nexl-builder 在 Meoo 上只能调用 Meoo 境内合规模型（qwen/kimi/glm/deepseek/MiniMax），**不得引导用户接 OpenAI 等境外模型**（违反规则⑩）。这与 Coze 国际版可走境外模型不同，务必区分。
