# Call for 5 Seed Templates
### 让独立站成为继短视频之后的下一个风口
### Make the independent site the next wave after short video

> **English**: If short video was the last democratization of expression, the independent site is the next democratization of ownership. Short video amplified the creator's voice but locked the audience's allegiance inside platform algorithms; the independent site puts the domain, the code, the brand narrative, and the user relationship back into your own hands. nexl lets anyone build a world-class site in a single conversation, and the template library is our shared digital asset — every scaffold you contribute becomes the starting point of someone else's compound growth. What we are doing is making the independent site the next wave after short video.

## 一、为什么是现在 · Why now

**中文（中本聪视角）**：一个有真正主权的存在，不能建立在可信第三方之上。你的品牌、你的观众、你的叙事，不该由别人的推荐引擎决定生死。短视频教会了世界"人人可表达"，却也证明了"平台即房东"的脆弱——一夜限流，资产归零。独立站是密码朋克精神在品牌维度的落地：点对点地拥有，而非恳求算法施舍。

**English (Satoshi's lens)**: A truly sovereign presence cannot be built on a trusted third party. Your brand, your audience, your narrative should not live or die by someone else's recommendation engine. Short video taught the world that anyone can express — yet it also proved the fragility of "the platform is the landlord": one algorithm change and the asset goes to zero. The independent site is cypherpunk philosophy applied to the dimension of brand: own it peer-to-peer, rather than begging the algorithm for alms.

**中文（乔布斯视角）**：我们生来是要在这个宇宙里留下痕迹的。一个品牌不该只是一张被刷过去的卡片，它该是一座你亲手建的数字据点。别人给你流量，我们给你据点。Insanely great 的独立站，不该只有大公司建得起——现在，任何人一次对话就能造出。

**English (Jobs' lens)**: We're here to put a dent in the universe. A brand should not be a card someone swipes past; it should be a digital stronghold you build with your own hands. Others give you traffic; we give you the stronghold. An insanely great independent site should not be the privilege of big companies alone — now, anyone can build one in a single conversation.

## 二、我们在建造什么 · What we are building

**中文**：nexl 是一套"双核"建站操作系统——先凝品牌灵魂（broker），再长成站点（builder），且每次迭代守护同一声音。模板库（`templates/`）是其中可复用的集体资产：每个模板带 machine-readable 的三维评级（quality / cost_efficiency / stability），被激活的 agent 会在建站时自动调出该平台最优骨架。你不是在从空白页起步，而是在巨人肩上起步。

**English**: nexl is a dual-core site-building OS — it first distills the brand soul (broker), then grows the site (builder), and guards the same voice on every iteration. The template library (`templates/`) is its reusable collective asset: every template carries a machine-readable 3-axis rating (quality / cost_efficiency / stability), and the activated agent automatically recalls the best scaffold for the platform when building. You don't start from a blank page; you start on the shoulders of giants.

**中文**：我们需要覆盖五个类目的种子模板，每个都是一类品牌的"世界观起点"。其中作品集已有示范（`aurora-minimal`），其余四类目开放给全球开发者共建：

**English**: We need seed templates across five categories, each a "worldview starting point" for a type of brand. Portfolio already has a reference (`aurora-minimal`); the other four are open for global developers to co-build:

| Category | Meaning | Status |
|---|---|---|
| portfolio | personal/studio showcase | ✅ aurora-minimal (reference) |
| brand-site | company/consumer brand home | 🟡 open call |
| landing | single-product conversion page | 🟡 open call |

## 四、如何贡献 · How to contribute

**中文**：
1. Fork `nexl-web-skills`，复制 `templates/_template/` 起手
2. 填 `template.json`（含三维 ratings 与 category）
3. 放 `site/` 源码（零依赖优先，暗黑极简工艺）
4. 提 PR —— 我们在 README 与 `INDEX.json` 收录你，author 即你的 GitHub handle

荣誉归你：每个模板的 star 与生态位，都刻着你的名字。

**English**:
1. Fork `nexl-web-skills`, copy `templates/_template/` to start
2. Fill `template.json` (with 3-axis ratings and category)
3. Add `site/` source (zero-dependency preferred, dark minimal craft)
4. Open a PR — we list you in README and `INDEX.json`, author = your GitHub handle

The honor is yours: every star and ecosystem position of a template bears your name.
