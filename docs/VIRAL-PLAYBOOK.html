二、nuwa 做对的 8 件事

  • 品类卡位：在"蒸馏人可行"窗口抢占更强叙事心智。
  • 低摩擦激活：npx skills add 一行装，跨 55+ runtime。
  • 效果可见即可传播：名人对话示例天然截图裂变。
  • 质量背书：保真度评分卡（盲测 A 级）解决信任。
  • UGC 飞轮：COMMUNITY 索引让用户提交 skill。
  • 多语言：中/英/日/韩/西吃全球。
  • 内容引擎：promo/hero/quote cards 持续供给。
  • AI 高频迭代：单日 11 提交打磨 SKILL.md。

四、nexl 病毒引擎：8 杠杆 + 1 飞轮

8 根增长杠杆

  • L1零安装漏斗：EdgeOne demo 页，输入品牌即生成预览+分享卡+激活语。
  • L2低摩擦激活：双激活语（安装型短句 + 即载型固定句）。
  • L3产出即传播：gen-brand-card.cjs 自动生成带回流链接分享卡+文案。
  • L4UGC 案例飞轮：showcase/ + CONTRIBUTING，用户案例即活广告。
  • L5品牌经纪人留存飞轮：brand-profile 越用越厚，切换成本递增。
  • L6多语言 × 多平台：中/英 README + 三平台原生。
  • L7结构化裂变激励：邀请得积分/模板。
  • L8内容引擎 + KOL：真实案例即内容，种子 KOL 首发。

差异化飞轮

用户建站 → 产出分享卡(裂变) → 案例进 showcase(社会证明)
　→ brand-profile 沉淀(留存) → 下次更懂品牌(复利)
　→ 更愿意邀请(激励) → 新用户进漏斗 → 循环加速

nuwa 只有前半段（建完即走）；我们有完整闭环。
