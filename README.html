tideshellbase 是观涛虾（TideShell）在 nexlbase 生态中的具身（embodiment），按世界顶级 Agent 工程范式（Anthropic / Google / OpenAI 终端 Agent）组织能力。

设计哲学

① 渐进式披露 —— 平时只持 index.yaml 极简索引，按需加载 SKILL.md，用完即抛。
② 操作诚实 —— 所有能力落到确定性 CLI，人/Agent 共享同一操作面，可复现调试。
③ 三层解耦 —— MCP(连接/认证) · CLI(原子执行) · Skills(SOP 编排)。
