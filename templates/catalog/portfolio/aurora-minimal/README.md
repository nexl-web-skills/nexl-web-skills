# Aurora · 极简作品集 / Aurora Minimal Portfolio

暗黑极简、强排版、零渐变、单一强调色。第一屏 5 秒说清「你是谁」。

## 设计语言 / Design language
- 背景 `#0A0A0A`，正文 `#F5F5F5`，强调色 `#FFD803`（克制使用）
- 系统字体栈，零外部依赖，即拷即用
- 超大字号姓名 + 一句定位 = 视觉锤

## 适用场景 / Best for
设计师、摄影师、独立开发者、顾问的个人作品集 / 个人 IP 官网。
