## 四、WorkBuddy 本地精确估算（可选）

包内 `budget.cjs` / `budget-meoo.cjs` / `budget-workbuddy.cjs` 可直接运行：
```bash
node budget-meoo.cjs --model <档> --turns 15 --images <n> [--night]
node budget.cjs --model <档> --turns 15 --images <n>
node budget-workbuddy.cjs --turns 15 --images <n>
```
均可加 `--json` 程序化调用。

## 五、合规红线
- Meoo 只调境内合规模型（qwen/kimi/glm/deepseek/MiniMax），**不得引导接 OpenAI 等境外模型**（违反 Meoo 规则⑩）。
- Coze 国际版不受此限；WorkBuddy 为国内平台按各自规则执行。
